Pirate's Code

🎯 Objective

Be the last pirate standing with dice remaining!

🎲 Setup

Each player starts with 5 dice. All players roll their dice secretly - only you can see your own dice.

⚔️ Gameplay

  • Bidding: On your turn, bid how many dice of a certain face value you think are on the table (across ALL players' dice combined).
  • Raising: Each bid must be higher than the last - either more dice, or the same quantity with a higher face value.
  • Challenge: If you think the current bid is too high, call "LIAR!" to challenge.

🌟 Wild Ones

Dice showing 1 are WILD and count as ANY face value (except when bidding specifically on 1s).

💀 Losing Dice

  • If the bid is TRUE (enough dice exist): The challenger loses a die.
  • If the bid is FALSE (not enough dice): The bidder loses a die.

The player who lost the die starts the next round.

🏆 Winning

The last player with dice remaining wins!

💡 Tips

  • Count your own dice and estimate what others might have.
  • Remember: roughly 1/3 of all dice will match any given face (including wilds).
  • Bluffing is part of the game - but don't get caught!
